# Frontend Mentor - Calculator app
The Calculator App is a simple web-based calculator that allows users to perform basic arithmetic operations. It's designed to provide a convenient tool for performing calculations quickly and easily.
 
 ![a beautiful calculator](https://res.cloudinary.com/dz209s6jk/image/upload/v1652199545/Challenges/xye51b9no1nkpuonov8r.jpg)
## Features

- Addition, subtraction, multiplication, and division operations.
- Clear button to reset the calculator.
- Responsive design for a seamless experience on different devices.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/calculator-app.git`
2. Navigate to the project directory: `cd calculator-app`
3. Open `index.html` in your web browser

### Usage

1. Click on the number buttons or use your keyboard to enter numbers.
2. Click on the operator buttons or use your keyboard to select the desired operation.
3. The calculator will display the result of the calculations in real-time.
4. To reset the calculator, click the "C" button or press the "Escape" key.
